## Instalação

##### CLONAR O REPOSITÓRIO 
```Alpine Abuild
git clone https://github.com/guiguicdd/wabase-md.git
cd wabase-md
```

##### INSTALAR AS DEPENDÊNCIAS 
```Alpine Abuild
npm i
```

##### INICIAR O WABASEMD (BOT/ROBÔ)
```Alpine Abuild
npm start
```

#####  CLONE THE REPOSITORY 
```Alpine Abuild
git clone https://github.com/guiguicdd/wabase-md.git
cd wabase-md
```

#####  INSTALL THE DEPENDENCIES 
```Alpine Abuild
npm i
```

#####  STARTS WABASEMD (BOT/ROBÔ)
